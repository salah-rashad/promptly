How to install the `promptly` package in your Dart project.

## Steps
<Steps>

1. Check if you have installed Dart

   ```dart title="Check Dart version" 
   dart --version
   ```
   > If you don't have Dart installed, follow the [official installation guide](https://dart.dev/get-dart).

2. Create a new `cli-app` project.

   ```dart title="Create a new project"
   dart create -t package 
   ```
3. Add the `promptly` package to your `pubspec.yaml` file.

   ```bash title="Add package to the project"
   dart pub dev:add promptly
   ```
</Steps>
